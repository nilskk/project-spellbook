Add your extension icons to this directory:

- icon-16.png (16x16 pixels) - Small icon for toolbar
- icon-48.png (48x48 pixels) - Medium icon for permissions and management page
- icon-128.png (128x128 pixels) - Large icon for Chrome Web Store

You can create these using any image editor (Photoshop, GIMP, etc.) or use online tools
like Canva or Pixlr. The images should be in PNG format with transparency for best results.

Alternative: Use simple placeholder images from https://placeholder.com/ during development.
